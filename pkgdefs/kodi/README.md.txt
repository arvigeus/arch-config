# Kodi

## Shortcuts

- `\`: toggle fullscreen
- `o`:  [Player process info](https://kodi.wiki/view/Player_process_info#Player_Process_Info)

## Upscaling:

- Go to `Settings -> Player -> Videos`
- Set Render Method to `DXVA`
- Activate `DXVA Video Super Resolution`
